## Common Architecture Patterns

### API Backend

```
   +-----------+      +--------+      +---------+
   |           |      |        |      |         |
   | API       +----->+ Lambda +----->+ DynamoDB|
   | Gateway   |      |        |      |         |
   |           |      |        |      |         |
   +-----------+      +--------+      +---------+
```

### Event Processing

```
   +-----------+      +--------+      +---------+
   |           |      |        |      |         |
   | S3 Event  +----->+ Lambda +----->+ S3      |
   | SNS/SQS   |      |        |      | DynamoDB|
   |           |      |        |      |         |
   +-----------+      +--------+      +---------+
```

### Stream Processing

```
   +-----------+      +--------+      +---------+
   |           |      |        |      |         |
   | Kinesis   +----->+ Lambda +----->+ Firehose|
   | DynamoDB  |      |        |      | Elastic |
   | Streams   |      |        |      | Search  |
   +-----------+      +--------+      +---------+
```
